# 本地电脑环境测试部署教程

## 环境要求

| 软件 | 版本 | 说明 |
|------|------|------|
| PHP | 7.4+ / 8.x | 需开启扩展：pdo_mysql、redis、bcmath、openssl、fileinfo、curl、mbstring |
| MySQL | 8.0+ | 字符集 utf8mb4 |
| Redis | 6.x+ | 用于缓存、分布式锁、WebSocket Pub/Sub |
| Nginx | 1.18+ | 用于反向代理 PHP 和 WebSocket |
| Composer | 2.x | PHP 依赖管理 |
| Node.js | 16+ | 仅用于 Web 管理后台构建 |
| 微信开发者工具 | 最新版 | 用于小程序调试 |

---

## 一、本地环境搭建

### 1.1 安装 PHP 及扩展

**macOS（Homebrew）：**
```bash
brew install php@8.0
brew install redis
brew install composer
```

**Windows（推荐使用 XAMPP 或 PHPStudy）：**
- 下载并安装 [PHPStudy](https://www.xp.cn/) 或 [XAMPP](https://www.apachefriends.org/)
- 确保 PHP 版本 ≥ 7.4

**Linux（Ubuntu/Debian）：**
```bash
sudo apt update
sudo apt install -y php8.0 php8.0-fpm php8.0-mysql php8.0-redis php8.0-bcmath php8.0-mbstring php8.0-curl php8.0-xml php8.0-fileinfo
sudo apt install -y redis-server mysql-server nginx composer
```

### 1.2 验证 PHP 扩展
```bash
php -m | grep -E "pdo_mysql|redis|bcmath|openssl|fileinfo|curl|mbstring"
```

## 二、数据库初始化

### 2.1 启动 MySQL
```bash
# macOS
brew services start mysql

# Linux
sudo systemctl start mysql
```

### 2.2 创建数据库
```bash
mysql -u root -p
```

执行以下 SQL：
```sql
CREATE DATABASE IF NOT EXISTS `game_platform` DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 2.3 导入表结构
```bash
mysql -u root -p game_platform < sql/schema.sql
```

## 三、后端部署

### 3.1 进入后端目录
```bash
cd php/
```

### 3.2 安装 Composer 依赖
```bash
composer install
```

### 3.3 配置环境变量
复制并编辑 `.env` 文件：
```bash
cp .env.example .env
```

编辑 `.env` 中的关键配置：
```ini
# 数据库
DATABASE_HOSTNAME = 127.0.0.1
DATABASE_DATABASE = game_platform
DATABASE_USERNAME = root
DATABASE_PASSWORD = your_password

# Redis
REDIS_HOST = 127.0.0.1
REDIS_PORT = 6379
REDIS_PASSWORD =

# JWT 密钥（请自行生成随机字符串）
JWT_SECRET = your_random_secret_key_here

# 微信小程序
WX_APP_ID = your_appid
WX_APP_SECRET = your_app_secret

# 邮件（SMTP）
SMTP_HOST = smtp.qq.com
SMTP_PORT = 465
SMTP_USER = your_email@qq.com
SMTP_PASS = your_smtp_password
```

### 3.4 启动 PHP 内置服务器（开发调试）
```bash
php -S 127.0.0.1:8080 -t public/
```

### 3.5 配置 Nginx（可选，推荐）
```nginx
server {
    listen 80;
    server_name localhost;
    root /path/to/php/public;
    index index.php;

    location / {
        if (!-e $request_filename) {
            rewrite ^(.*)$ /index.php?s=$1 last;
        }
    }

    location ~ \.php$ {
        fastcgi_pass 127.0.0.1:9000;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        include fastcgi_params;
    }
}
```

### 3.6 启动 WebSocket 服务
```bash
php think worker:start
```

> 默认监听 `0.0.0.0:2345`，可在 `config/worker_server.php` 中修改。

## 四、Web 管理后台部署

### 4.1 进入目录
```bash
cd admin-web/
```

### 4.2 安装依赖
```bash
npm install
```

### 4.3 配置代理
编辑 `vite.config.js`，确保 API 代理指向后端地址：
```js
server: {
  proxy: {
    '/api': {
      target: 'http://127.0.0.1:8080',
      changeOrigin: true
    }
  }
}
```

### 4.4 启动开发服务器
```bash
npm run dev
```

浏览器访问 `http://localhost:5173` 即可看到登录页面。

### 4.5 默认管理员账号
| 账号 | 密码 |
|------|------|
| admin | 1234567 |
| admin2 | 1234567 |

> 首次登录将强制要求修改密码并绑定邮箱。

## 五、微信小程序调试

### 5.1 打开微信开发者工具
1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 使用微信扫码登录

### 5.2 导入项目
1. 点击 "导入项目"
2. 项目目录选择 `WX-xcx/`
3. AppID 填写你自己的小程序 AppID（或使用测试号）
4. 点击 "导入"

### 5.3 配置后端地址
编辑 `WX-xcx/utils/request.js` 中的 `BASE_URL`：
```javascript
const BASE_URL = 'http://127.0.0.1:8080/api/v1';
```

> 注意：小程序开发时需要勾选 "不校验合法域名" 选项

### 5.4 配置 WebSocket 地址
编辑 `WX-xcx/utils/websocket.js` 中的 WebSocket URL：
```javascript
const WS_URL = 'ws://127.0.0.1:2345';
```

## 八、常见问题

### Q: 数据库连接失败？
检查 MySQL 服务是否启动，`.env` 中的数据库配置是否正确。

### Q: Redis 连接失败？
检查 Redis 服务是否启动：`redis-cli ping` 应返回 `PONG`。

### Q: WebSocket 连接失败？
检查 Workerman 是否启动成功：`php think worker:status`。

### Q: 小程序无法请求本地 API？
微信开发者工具中勾选 "详情 → 本地设置 → 不校验合法域名"。

### Q: Composer 安装慢？
使用国内镜像：
```bash
composer config -g repo.packagist composer https://mirrors.aliyun.com/composer/
```

### Q: npm 安装慢？
使用国内镜像：
```bash
npm config set registry https://registry.npmmirror.com
```
