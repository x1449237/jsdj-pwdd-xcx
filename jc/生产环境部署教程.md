# 生产环境部署教程

## 环境要求

| 软件 | 版本 | 说明 |
|------|------|------|
| 操作系统 | CentOS 7+ / Ubuntu 18.04+ / Debian 10+ | 64位 |
| PHP | 7.4+ / 8.x | 推荐 8.0 |
| MySQL | 8.0+ | 主从复制推荐 |
| Redis | 6.x+ | 哨兵/集群模式推荐 |
| Nginx | 1.20+ | 反向代理 + 静态资源 |
| Supervisord | 4.x | 进程守护 |
| Node.js | 16+ | 仅用于构建前端 |

## 二、数据库配置

### 2.1 MySQL 安全初始化
```bash
mysql_secure_installation
```

### 2.2 创建数据库
```sql
CREATE DATABASE IF NOT EXISTS `game_platform` DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER IF NOT EXISTS 'game_platform'@'localhost' IDENTIFIED BY 'your_strong_password';
GRANT ALL PRIVILEGES ON game_platform.* TO 'game_platform'@'localhost';
FLUSH PRIVILEGES;
```

### 2.3 导入表结构
```bash
mysql -u game_platform -p game_platform < /var/www/project/sql/schema.sql
```

### 2.4 MySQL 性能优化配置
编辑 `/etc/my.cnf` 或 `/etc/mysql/my.cnf`：
```ini
[mysqld]
innodb_buffer_pool_size = 2G
innodb_log_file_size = 512M
innodb_flush_log_at_trx_commit = 2
max_connections = 500
query_cache_type = 0
slow_query_log = 1
slow_query_log_file = /var/log/mysql/slow.log
long_query_time = 2
```

## 四、后端部署

### 4.1 创建项目目录
```bash
mkdir -p /var/www/project
cd /var/www/project
```

将 `php/` 目录下的所有文件上传到 `/var/www/project/php/`。

### 4.2 设置权限
```bash
chown -R www-data:www-data /var/www/project/php/
chmod -R 755 /var/www/project/php/
chmod -R 777 /var/www/project/php/runtime/
chmod -R 777 /var/www/project/php/public/uploads/
```

### 4.3 安装依赖
```bash
cd /var/www/project/php/
composer install --no-dev --optimize-autoloader
```

### 4.4 配置环境变量
```bash
cp .env.example .env
vim .env
```

关键配置项：
```ini
APP_DEBUG = false

DATABASE_HOSTNAME = 127.0.0.1
DATABASE_DATABASE = game_platform
DATABASE_USERNAME = game_platform
DATABASE_PASSWORD = your_strong_password

REDIS_HOST = 127.0.0.1
REDIS_PORT = 6379
REDIS_PASSWORD = your_redis_password

JWT_SECRET = generate_a_64_char_random_string_here

WX_APP_ID = your_real_appid
WX_APP_SECRET = your_real_app_secret

SMTP_HOST = smtp.qq.com
SMTP_PORT = 465
SMTP_USER = your_email@qq.com
SMTP_PASS = your_smtp_password
```

### 4.5 生成 JWT 密钥
```bash
php -r "echo bin2hex(random_bytes(32));"
```
将输出的字符串填入 `.env` 的 `JWT_SECRET`。

## 五、Nginx 配置

### 5.1 后端 API 配置
创建 `/etc/nginx/conf.d/game-platform.conf`：
```nginx
# 后端 API
server {
    listen 80;
    server_name api.yourdomain.com;
    root /var/www/project/php/public;
    index index.php;

    # 日志
    access_log /var/log/nginx/api_access.log;
    error_log /var/log/nginx/api_error.log;

    # 上传限制
    client_max_body_size 50M;

    # 安全头
    add_header X-Frame-Options "SAMEORIGIN" always;
    add_header X-Content-Type-Options "nosniff" always;
    add_header X-XSS-Protection "1; mode=block" always;

    location / {
        if (!-e $request_filename) {
            rewrite ^(.*)$ /index.php?s=$1 last;
        }
    }

    location ~ \.php$ {
        fastcgi_pass unix:/var/run/php/php8.0-fpm.sock;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        fastcgi_param PHP_VALUE "open_basedir=/var/www/project/php/:/tmp/";
        include fastcgi_params;
        fastcgi_read_timeout 300;
    }

    # 禁止访问敏感文件
    location ~ /\. {
        deny all;
    }
    location ~ /(runtime|vendor)/ {
        deny all;
    }
}

# WebSocket 代理
server {
    listen 80;
    server_name ws.yourdomain.com;

    location / {
        proxy_pass http://127.0.0.1:2345;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_read_timeout 86400s;
        proxy_send_timeout 86400s;
    }
}

# Web 管理后台
server {
    listen 80;
    server_name admin.yourdomain.com;
    root /var/www/project/admin-web/dist;
    index index.html;

    access_log /var/log/nginx/admin_access.log;
    error_log /var/log/nginx/admin_error.log;

    location / {
        try_files $uri $uri/ /index.html;
    }

    # API 反向代理
    location /api/ {
        proxy_pass http://127.0.0.1:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}
```

### 5.2 启动 Nginx
```bash
nginx -t
systemctl enable nginx
systemctl restart nginx
```

## 七、WebSocket 进程守护

### 7.1 安装 Supervisor
```bash
# CentOS
yum install -y supervisor

# Ubuntu
apt install -y supervisor
```

### 7.2 配置 Supervisor
创建 `/etc/supervisor/conf.d/game-platform.conf`：
```ini
[program:game-platform-worker]
command=php /var/www/project/php/think worker:start
directory=/var/www/project/php/
user=www-data
autostart=true
autorestart=true
startretries=10
redirect_stderr=true
stdout_logfile=/var/log/game-platform-worker.log
stdout_logfile_maxbytes=50MB
stdout_logfile_backups=10
```

### 7.3 配置定时任务
创建 `/etc/supervisor/conf.d/game-platform-cron.conf`：
```ini
[program:game-platform-timeout]
command=php /var/www/project/php/think order:timeout
directory=/var/www/project/php/
user=www-data
autostart=true
autorestart=true
stdout_logfile=/var/log/game-platform-timeout.log

[program:game-platform-settle]
command=php /var/www/project/php/think order:settle
directory=/var/www/project/php/
user=www-data
autostart=true
autorestart=true
stdout_logfile=/var/log/game-platform-settle.log

[program:game-platform-probe]
command=php /var/www/project/php/think player:probe
directory=/var/www/project/php/
user=www-data
autostart=true
autorestart=true
stdout_logfile=/var/log/game-platform-probe.log

[program:game-platform-backup]
command=php /var/www/project/php/think backup:daily
directory=/var/www/project/php/
user=www-data
autostart=true
autorestart=true
stdout_logfile=/var/log/game-platform-backup.log

[program:game-platform-retry]
command=php /var/www/project/php/think third:retry
directory=/var/www/project/php/
user=www-data
autostart=true
autorestart=true
stdout_logfile=/var/log/game-platform-retry.log

[program:game-platform-reminder]
command=php /var/www/project/php/think appeal:reminder
directory=/var/www/project/php/
user=www-data
autostart=true
autorestart=true
stdout_logfile=/var/log/game-platform-reminder.log

[program:game-platform-monitor]
command=php /var/www/project/php/think monitor:alert
directory=/var/www/project/php/
user=www-data
autostart=true
autorestart=true
stdout_logfile=/var/log/game-platform-monitor.log
```

### 7.4 启动 Supervisor
```bash
supervisorctl reread
supervisorctl update
supervisorctl start all
```

### 7.5 常用 Supervisor 命令
```bash
supervisorctl status                    # 查看所有进程状态
supervisorctl restart game-platform-worker  # 重启 WebSocket
supervisorctl tail -f game-platform-worker  # 查看日志
```

## 十、安全加固

### 10.1 数据库安全
- 使用强密码（≥16位，包含大小写字母、数字、特殊字符）
- 限制数据库仅允许本地连接：`bind-address = 127.0.0.1`
- 定期备份

### 10.2 Redis 安全
- 设置 `requirepass` 强密码
- 绑定 `bind 127.0.0.1`
- 禁用危险命令：`rename-command FLUSHDB ""`

### 10.3 PHP 安全
```ini
# /etc/php/8.0/fpm/php.ini
expose_php = Off
display_errors = Off
log_errors = On
allow_url_fopen = Off
disable_functions = exec,passthru,shell_exec,system,proc_open,popen,curl_exec,curl_multi_exec,parse_ini_file,show_source
```

### 10.4 文件权限
```bash
# 所有文件属主为 www-data
chown -R www-data:www-data /var/www/project/

# 目录 755，文件 644
find /var/www/project/ -type d -exec chmod 755 {} \;
find /var/www/project/ -type f -exec chmod 644 {} \;

# runtime 和 uploads 需要写权限
chmod -R 777 /var/www/project/php/runtime/
chmod -R 777 /var/www/project/php/public/uploads/
```

### 10.5 配置 NTP 时间同步
```bash
# 安装 chrony
yum install -y chrony  # CentOS
apt install -y chrony  # Ubuntu

# 启动
systemctl enable chronyd
systemctl start chronyd

# 检查同步状态
chronyc tracking
```

## 十一、备份策略

### 11.1 数据库备份脚本
创建 `/opt/scripts/backup.sh`：
```bash
#!/bin/bash
DATE=$(date +%Y%m%d_%H%M%S)
BACKUP_DIR="/var/backups/game_platform"
mkdir -p $BACKUP_DIR

# 数据库备份
mysqldump -u game_platform -p'your_password' game_platform | gzip > $BACKUP_DIR/db_$DATE.sql.gz

# 加密备份
openssl enc -aes-256-cbc -salt -in $BACKUP_DIR/db_$DATE.sql.gz -out $BACKUP_DIR/db_$DATE.sql.gz.enc -k 'your_encryption_key'

# 删除原始文件
rm $BACKUP_DIR/db_$DATE.sql.gz

# 保留最近30天
find $BACKUP_DIR -name "*.enc" -mtime +30 -delete
```

### 11.2 添加定时任务
```bash
chmod +x /opt/scripts/backup.sh
crontab -e
```

添加：
```
0 2 * * * /opt/scripts/backup.sh
```

## 十五、故障排查

### 常见问题

**Q: 502 Bad Gateway？**
检查 PHP-FPM 是否运行：`systemctl status php8.0-fpm`

**Q: WebSocket 连接不上？**
检查 Workerman 进程：`supervisorctl status game-platform-worker`

**Q: 小程序请求超时？**
1. 检查域名是否配置正确
2. 检查 SSL 证书是否有效
3. 检查防火墙是否开放 443 端口

**Q: 数据库连接慢？**
1. 检查慢查询日志
2. 优化索引
3. 考虑读写分离

**Q: 提现或支付异常？**
1. 检查 Redis 是否运行
2. 检查分布式锁是否正常释放
3. 查看后台日志定位问题
